# Data Automation Pipeline

## Overview
Automated pipeline that collects, cleans and analyzes data from a public API.

## Features
- automated data collection
- data cleaning pipeline
- report generation
- scheduled execution

## Architecture
API → Data Collection → Data Cleaning → Report Generation

## Tech Stack
- Python
- Pandas
- Requests
- Schedule

## How to Run
pip install -r requirements.txt
python src/main.py
